= tmux

Provides basic tmux integration for Kakoune.

== Usage

Enable tmux integration by opening a client inside tmux, then use the window commands.

== Commands

- `tmux`
- `tmux-focus`
- `tmux-terminal-horizontal`
- `tmux-terminal-vertical`
- `tmux-terminal-tab`
- `tmux-terminal-popup`
- `tmux-terminal-panel`

== Aliases

- `terminal`
- `terminal-horizontal`
- `terminal-vertical`
- `terminal-tab`
- `popup`
- `panel`
- `focus`
