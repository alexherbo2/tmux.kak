# tmux.kak

Provides basic [tmux] integration for [Kakoune].

[Kakoune]: https://kakoune.org
[tmux]: https://github.com/tmux/tmux

## Installation

``` sh
git clone https://github.com/alexherbo2/tmux.kak.git ~/.config/kak/autoload/tmux
```
